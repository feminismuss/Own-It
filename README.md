# Own It

Own It ist ein kollaboratives Planner-Board für gemeinsame 
Aufgaben und Projekte. Die App hilft Gruppen dabei, Aufgaben 
sichtbar zu machen, sie aufzuteilen und den Fortschritt im 
Blick zu behalten.

## Features

### MVP
- Todos erstellen, sehen, editieren und löschen
- Status eines Todos setzen: To Do, In Progress oder Done
- Projekte: Jedes Projekt hat eine eigene Todo-Liste
- Projekte anlegen, editieren und löschen
- Login: Nutzer können sich registrieren und einloggen

### Nice to Have
- Kanban-Ansicht mit Drag & Drop
- Projekten per Invite-Link joinen
- "Ich mach das!" – Aufgabe übernehmen und automatisch in In Progress setzen

## Tech Stack

- **Library:** React
- **Framework:** Next.js
- **Datenbank:** MongoDB
- **CSS:** Styled Components
- **Auth:** NextAuth.js

## Getting Started

1. Repository klonen:
\```bash
git clone https://github.com/dein-username/own-it.git
\```

2. Abhängigkeiten installieren:
\```bash
npm install
\```

3. Entwicklungsserver starten:
\```bash
npm run dev
\```

Die App läuft dann unter [http://localhost:3000](http://localhost:3000)
